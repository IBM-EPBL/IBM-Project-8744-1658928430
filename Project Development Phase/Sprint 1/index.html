Handwritten Recognition System

Handwritten Test Recognition is a technology that is much needed in this world as of today. This Digit Recognition System is used to recognise the digits from different sources like emails, papers, images etc. Before proper implementation od this technology we have relied on writing texts with our own hands which can result in errors. It's difficult to store and access physical data with efficiency. The project presents recognising the handwritten digits (0 to 9) from the famous MNIST dataset. Here, we will be using artificial neural networks/ convolution neural network.
